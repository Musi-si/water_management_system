Water Management System

This is a system for monitoring water usage, with admin and client accounts.
For admin signup and login a PIN is required to prove that the user is an admin.
After logging in as an admin, one can view all admins and all clients registered.
The admin is also capable of deleting an admin or client from the system.
They can also filter these records using different criteria like date and quantity.
Client signup and login is simpler and does not require the PIN.
They can view their usage records as a table or a graph and they can insert usage data.
They are also capable of filtering the data using dates and quantity.
